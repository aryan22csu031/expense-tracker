*Guidelines for CSV File Input*

File Format:
Ensure your file is in CSV format. Only CSV files are accepted.
Column Names:
Your CSV file must contain the following columns with exact names:
  • Date: This column should contain the date of each transaction. Ensure all dates are in a consistent format, such as YYYY-MM-DD.
  • Category: This column should describe the type of transaction (e.g., "Groceries", "Salary", "Utilities").
  • Withdrawal: This column should contain amounts withdrawn from your account.
  • Deposit: This column should contain amounts deposited into your account.
Check for Missing Data:
Ensure there are no missing values in the required columns.
Suggestions:
If your file is not in CSV format:
Open your file in a spreadsheet application (like Microsoft Excel or Google Sheets). Save or export the file as a CSV (Comma Separated Values) file.

If your file does not contain the exact column names:
Use the following prompt in ChatGPT to convert your file into the desired format:
"I have a file containing transaction details. The columns are not named as 'Date', 'Category', 'Withdrawal', and 'Deposit'. Please provide a step-by-step guide on how to rename my columns to these exact names and ensure the data is correctly formatted for upload to Bachat Buddy."
